O que se trata a linguagem de programação?

(retornar ao índice)

A linguagem de programação é, basicamente, a linguagem da máquina. É um método padronizado para instruir uma máquina por meio de códigos dos mais variados. Ela serve para que seja possível a "tradução" da linguagem natural para a artifical.

Existem diversos tipos de linguagens de programação, onde cada uma exerce sua função na máquina. algumas funcionam mais para o lado dos servidores (Server-side), já outras para o lado do cliente — ou usuários de um website (Client-side).

Veja, a seguir, do que se tratam as linguagens server-side e client-side:


Linguagem Server-side

(retornar ao índice)

As linguagens server-side são aquelas de entendimento do servidor, ou seja, ele quem vai entender as informações do código e fazer o envio para o navegador. Elas também podem ser denominadas de back-end.

Alguns exemplos de linguagens server-side são:

  • PHP;
  • Java;
  • C#;
  • JavaScript.

Por fim, aqueles que trabalham projetando os códigos para o lado do servidor são denominados Desenvolvedores Back-end.


Linguagem Client-side

(retornar ao índice)

As linguagens client-side, por outro lado, são aquelas de entendimento do navegador web, ou seja, é ele quem vai compreender as instruções de código e mostrar ao cliente/usuário. Elas também podem ser denominadas de front-end.

Alguns exemplos de linguagens client-side:

  • TypeScript;
  • JavaScript.

Por fim, aqueles que trabalham projetando os códigos para o lado do cliente são denominados Desenvolvedores Front-End.


Extra: Afinal, HTML é ou não é linguagem de programação?

Bem, HTML não é uma linguagem de programação. Como a própria sigla HTML diz, ela é uma linguagem de marcação de hipertexto. Assim como o CSS, eles apenas servem para a formatação de páginas e sites. As linguagens de programação são usadas para fins de entregar instruções que serão executadas dentro da máquina (computador).
